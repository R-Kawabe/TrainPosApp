# 電車検索サービスのサンプル

## 実行方法

```
$> php -S 0.0.0.0:8000
```

http://localhost:8000 にアクセスすると、開ける

- ページレイアウトなど
- CSS
- 右側にサイドバー追加（検索フォーム、他ページリンク）
- それかヘッダーに他ページリンク？
